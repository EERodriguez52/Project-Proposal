Application Overview

FireteamFormations will be a quick, easy to use guide to help visualize the vaarious formations used at the fireteam level. This will help teach the formations, and their applications in a classroom environment.

Technology Overview

Using a combination of HTML, CSS, and JavaScript, the application will use green sprites, to represent a fireteam of Marines, and then when the corresponding button is clicked, the Marines will arrange themselves in the selected formation.

Proposal Last Updated

2026/04/08

Feature List

The features that will be deliverable will be the ability to select the formation desired, and then watch the "Marines" ararnge themselves accordingly. I also plan to include icons for each member of the fireteam, designating the Team Leader, Grenadier, Automatic Rifleman, and the Rifleman.

Future Feature List

If possible, I would like to include a demonstration of a rocket battle drill or a frag battle drill. I would also like to include arrows that move with the Marines, which would illustrate their fields of fire.

Basic Technical Features

The basic features would be css styling, and basic button functions.

Advanced Technical Features

Advanced technical features include animating the sprites, including icons for each sprite, and corresponding each formation to a button.
